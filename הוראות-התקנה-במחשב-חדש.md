# איך להמשיך לעבוד על האתר של הדס במחשב חדש

קובץ זה מסכם איך מעבירים את כל הסביבה למחשב אחר (גם מחשב שאינו מק).

---

## איך זה בנוי — שני חלקים

1. **קלוד** — העוזר שאיתו עובדים. הוא לא מחזיק את העבודה בתוכו.
2. **קבצי האתר** — שמורים ב־GitHub (בענן), בריפו `Tsookn/hadas`.
   צריך להוריד אותם לכל מחשב חדש. זה לא קורה לבד.

חשוב: גם השיחות עם קלוד **לא** עוברות למחשב אחר. במחשב החדש קלוד לא
"יזכור" שיחות קודמות — אבל הוא יראה את הקבצים, וזה מה שחשוב.

---

## שלבים במחשב חדש

### 1. להוריד את קבצי האתר
- להתקין את GitHub Desktop מהכתובת: desktop.github.com
- להתחבר עם החשבון `Tsookn`
- ללחוץ על שכפול (Clone) של הריפו `hadas` — זה מוריד את התיקייה למחשב

### 2. להתקין את קלוד
- אפשרות א' (בדפדפן): להיכנס ל־claude.ai/code ולהתחבר עם אותו חשבון.
- אפשרות ב' (אפליקציה): להוריד את התוכנה למחשב (יש גם לחלונות, לא רק למק).

### 3. לחבר את קלוד לקבצים
כשקלוד מציע שלוש דרכים — טרמינל / רשת / VS — לבחור **VS** (זה VS Code).
- אם צריך, קודם להתקין את VS Code מהכתובת: code.visualstudio.com
- בתוך VS Code: ‏`File → Open Folder` ולבחור את תיקיית `hadas` שירדה.
- עכשיו קלוד רואה את כל הקבצים, וממשיכים מכאן.
